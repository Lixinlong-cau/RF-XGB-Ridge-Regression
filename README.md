# RF-XGB-Ridge-Regression
A Stacking ensemble learning model integrating Random Forest, Extreme Gradient Boosting, and Ridge Regression

Code repository for reproducing analyses and figures in the manuscript:
**Regional Adaptation and Optimization of Wheat Field Management Practices in China Based on Stacking Ensemble Learning**

## Overview
This repository contains the custom code used for **GridSearch-optimized stacking ensemble learning modeling**, feature importance analysis, and prediction framework to investigate wheat yield (YD), water use efficiency (WUE), and optimized agricultural management practices in China.

## Repository Structure
├── X_train.xlsx        # Training set feature data

├── Y_train_YD.xlsx     # Training set yield label

├── Y_train_WUE.xlsx    # Training set WUE label

├── X_test.xlsx         # Test set feature data

├── Y_test_YD.xlsx      # Test set yield label

├── Y_test_WUE.xlsx     # Test set WUE label

└── main.py             # Main program script


## Reproducibility Instructions
- **Main Prediction Results & Figures**  
  Generated by running scripts in code.py, including prediction comparison plots and feature importance ranking plots.
- **Prediction and Feature Importance Tables**  
  Directly generated by running core scripts and saved.

## Core Functions
- Label encoding and standardization for categorical and numerical features
- GridSearchCV + 5-fold cross-validation for hyperparameter optimization
- Stacking ensemble training and prediction (RF + XGB as base models, Ridge as meta-model)
- Weighted average ensemble for performance comparison
- Feature importance calculation and visualization
- Model evaluation (R², RMSE, error analysis)
- Automated result output and visualization

## Usage
- Place training and test Excel files in the root directory
- Switch prediction target (YD / WUE) by modifying file path annotations
- Run the main script in code.py
- Check all outputs
